# Es-Magico | MERN Stack Project Management Tool

Es-Magico is a robust Project Management Dashboard designed for seamless project creation and real-time tracking. Built with the MERN stack, it features secure authentication and a dynamic user interface.

## 🛠️ Technical Stack

**Frontend:**
- **React.js (Vite):** Core framework for a fast, component-based UI.
- **Tailwind CSS:** For modern, responsive, and utility-first styling.
- **Axios:** For handling asynchronous API requests to the backend.
- **Lucide React:** For consistent and lightweight iconography.
- **Framer Motion:** For smooth UI transitions and animations.

**Backend:**
- **Node.js & Express.js:** Scalable server-side environment and routing.
- **MongoDB Atlas:** Cloud-based NoSQL database for flexible data storage.
- **JWT (JSON Web Tokens):** Secure, stateless user authentication.
- **Mongoose:** ODM for schema-based data modeling.

## ✨ Key Features

- **User Authentication:** Secure Signup and Login functionality with protected routes.
- **Dynamic Dashboard:** A centralized view to manage and track all created projects.
- **Project Creation:** An interactive, modal-based system to add new projects instantly.
- **Fully Responsive:** Optimized for a seamless experience across Mobile, Tablet, and Desktop.
- **Real-time Data Sync:** Immediate UI updates upon successful backend operations.
